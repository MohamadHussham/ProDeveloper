# Mohamad Hamdan — Portfolio (multi-page repo)

This repository is a starter multi-page portfolio for a front-end developer.
It includes:
- `index.html` — main portfolio homepage
- `services.html`, `contact.html` — supporting pages
- `projects/` — individual project pages
- `styles.css`, `script.js` — shared assets
- `assets/` — sample images and favicon

## How to use
1. Edit the HTML files to replace content and screenshots in `assets/`.
2. Update `styles.css` with your branding (colors, fonts).
3. For deploying:
   - GitHub Pages: push to a public repo and enable Pages.
   - Netlify: drag & drop the repo folder to Netlify or connect via Git.
4. To add a backend for the contact form, replace the mock handler in `script.js` with your API endpoint.
